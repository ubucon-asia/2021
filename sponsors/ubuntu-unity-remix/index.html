Ubuntu Unity Remix

Ubuntu Unity Remix is an Ubuntu remix (unofficial flavour) which uses Unity7 instead of gnome-shell. The latest stable release of Ubuntu Unity is 21.04 and we are also working on the desktop successor of Unity7, UnityX.

  • Web: https://ubuntuunity.org
  • Twitter: https://twitter.com/ubuntu_unity
  • Telegram: https://t.me/ubuntuunityannouncements
  • Discord: https://discord.gg/M2AWuCK
