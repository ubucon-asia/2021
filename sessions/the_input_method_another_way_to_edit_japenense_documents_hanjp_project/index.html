The input method, another way to edit Japenense documents, Hanjp project

2021-09-26 01:15 PM +0900

Room 1
Session duration 30m0s

In this session, I’d like to introduce about Hanjp Input Method project, first of all. Its concept, the reasons why did I start the project, current state, and some demos are included. Then, I will re-introduce it in techincal view. Choosen languages and libraries, program structure, and project milestone are included. Then lastly, I’d talk about community development.
